# S Mohammed Junaid — Portfolio

Premium personal portfolio website. Pure HTML · CSS · JavaScript. No frameworks. No build tools.

---

##  How to Open

1. **Download** and **unzip** the folder
2. **Double-click** `index.html` → opens in your browser
3. Done. 

> Works in Chrome, Firefox, Edge, and Safari. Requires internet for Google Fonts.

---

##  Features

- Particle network canvas (mouse-interactive)
- Custom 2-layer cursor with blend mode
- Typewriter cycling role animation
- 3D tilt on skill & project cards
- Magnetic CTA buttons
- Horizontal drag-to-scroll projects
- Animated page loader
- Scroll progress bar
- Scroll-reveal animations (up / left / right)
- Sticky experience layout
- Tech marquee ticker
- Count-up stat numbers
- Fully responsive (mobile-friendly)
- SEO meta tags + Open Graph
